Parsing avito.ru phone
========================
1. sudo apt-get install tesseract-ocr
2. wget http://pkgs.fedoraproject.org/repo/pkgs/tesseract-langpack/tesseract-ocr-3.02.rus.tar.gz
3. tar -zxvf tesseract-ocr-3.02.rus.tar.gz
4. cd tesseract-ocr/tessdata
5. sudo cp * /usr/share/tesseract-ocr/tessdata/
6. composer install
